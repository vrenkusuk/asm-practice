# Практична 2: Взаємодія з ОС

## Що таке int 0x80
`int 0x80` — системний виклик у Linux для x86. Використовується для взаємодії програми з ядром.

## Найпоширеніші функції
| Функція | Номер | Параметри | Опис |
|---------|-------|-----------|------|
| exit    | 1     | ebx = код виходу | Завершення програми |
| fork    | 2     | немає          | Створення нового процесу |
| read    | 3     | ebx = fd, ecx = buf, edx = count | Читання з файлу/вхідного потоку |
| write   | 4     | ebx = fd, ecx = buf, edx = count | Запис у файл/термінал |
| open    | 5     | ebx = path, ecx = flags, edx = mode | Відкрити файл |
| close   | 6     | ebx = fd       | Закрити файл |
| waitpid | 7     | ebx = pid, ecx = status, edx = options | Очікування завершення процесу |
